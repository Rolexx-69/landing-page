# Landing Page - SAKTHU Marketplace

A beautiful landing page built from Figma design for SAKTHU, a marketplace for buying and selling akampets.

## Features

- Responsive design matching Figma specifications
- Modern gradient background
- Interactive navigation
- Hero section with call-to-action buttons
- Featured logos section
- Analytics section
- App download section
- Testimonials section
- Newsletter signup
- Smooth scroll animations

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts (Outfit, Playfair Display, Roboto)

## Getting Started

Simply open `index.html` in your web browser to view the landing page.
